# my-admin(vue3)🐣



通用后台使用的技术👉vue-cli + vue3 + element-plus

学习经典的vue-admin后台项目



------



## 构建项目📡

```
# 克隆项目
git clone https://github.com/jzfai/vue3-admin-plus.git
```



## 项目安装🔄
```
npm install
&
yarn install
```



## 项目服务启动▶️

```
npm run serve
&
yarn serve
```



启动访问 ➡️ http://localhost:8080
